**Labour Bridge**

Labour Bridge is a web-based platform that connects labourers directly with employers/contractors, reducing dependency on middlemen and ensuring transparent, verified hiring.

**Problem Statement**
The unorganized labour sector lacks a reliable digital platform for connecting workers with employers. This results in exploitation, unverified hiring, and inefficient job matching.

**Solution**
Labour Bridge provides a role-based system where labourers, employers, and admins interact through a secure and centralized platform.

**User Roles**
- Labourer: Create profile, add skills, view and accept job requests
- Employer: Post job requirements, search skilled labourers
- Admin: Verify users, manage platform data

**Features**
- Role-based authentication
- Verified user profiles
- Skill-based labour search
- Job posting and hiring workflow

**Tech Stack**
- Frontend: HTML, CSS
- Backend: Spring Boot 
- Database: MySQL
